# FinNews Monitor — 金融新闻监控助手

> 自动监控 20 个金融媒体，根据您的持仓智能筛选相关新闻，通过邮件分级推送。

---

## 快速启动

```bash
# 1. 安装依赖（首次）
npm install

# 2. 编译并启动
npm start

# 然后访问 http://localhost:3000
```

**开发模式：**
```bash
npm run dev          # 直接运行（不编译）
npm run dev:watch    # 监听文件变化自动重启
```

## 使用流程

### 1. 添加持仓
访问 **持仓管理** 页面，添加您的持仓标的：
- 填写持仓名称（如：宁德时代）
- 填写证券代码（如：300750.SZ）
- 选择市场和类型（股票/ETF/基金）
- **填写关键词**（逗号分隔），或用"快速添加"按钮预设
- 可选：填写排除关键词（减少噪音）
- 可选：启用 AI 分析（推荐，需配置 AI 服务）
- 可选：设置邮件提醒阈值（默认 75 分）

### 2. 配置邮件
访问 **系统设置** → **邮件配置**：

**QQ邮箱配置示例：**
| 字段 | 值 |
|------|-----|
| SMTP 服务器 | `smtp.qq.com` |
| SMTP 端口 | `465` |
| SSL/TLS | 勾选 |
| 邮箱账号 | `你的QQ邮箱@qq.com` |
| SMTP 授权码 | 在 QQ邮箱设置→账户→SMTP 中获取 |
| 通知邮箱 | 接收邮件的邮箱地址 |

配置完成后点击 **发送测试邮件** 验证是否正常。

### 3. 配置 AI 分析（可选但推荐）
访问 **系统设置** → **AI 分析配置**，支持三种 AI 服务：

#### 🏠 OpenCode 本地（推荐，免费）
1. 安装 OpenCode CLI：`npm install -g opencode-ai`
2. 在设置页面选择 "OpenCode 本地"
3. 可选：手动指定 CLI 路径（留空自动检测）
4. 选择模型（推荐 "自动选择"）

**优势：** 完全免费、无 API 调用限制、不受网络限流影响

#### ☁️ OpenRouter（多模型选择）
1. 访问 [https://openrouter.ai/](https://openrouter.ai/) 注册账号
2. 进入 Settings → API Keys → 创建 Key
3. 在设置页面选择 "OpenRouter"，填入 API Key
4. 选择模型（推荐 **DeepSeek Chat** 免费版，中文理解好）

#### ☁️ Ollama 云端
1. 在设置页面选择 "Ollama 云端"
2. 配置 API Key 和地址
3. 选择模型

配置完成后点击 **测试 AI 连通性** 验证是否正常。

### 4. 开始监控
- 首次使用可在首页点击 **立即采集** 手动触发
- 系统会每 30 分钟自动采集一次（可配置）
- 每天 08:00、12:00、18:00 自动发送汇总邮件（可配置）
- 高影响新闻（AI 评分 ≥ 邮件提醒阈值）**实时推送**

## 项目结构

```
src/
├── app.ts              # Express 主入口
├── config/
│   ├── database.ts     # 数据存储配置（无需外部数据库）
│   └── news-sources.ts # 20个新闻源配置
├── modules/
│   ├── fetcher/        # 新闻采集（HTML/RSS/API 解析）
│   ├── analyzer/       # 关键词匹配 + AI 分析
│   │   └── opencode-client.ts  # OpenCode 本地客户端
│   ├── notifier/       # 邮件发送（nodemailer）
│   └── scheduler/      # 定时任务（node-schedule）
├── routes/             # Express 路由
├── views/              # EJS 模板
├── types/              # TypeScript 类型定义
└── utils/              # 工具函数
```

## 技术栈

- **后端框架**: Express.js + TypeScript
- **模板引擎**: EJS
- **数据存储**: JSON 文件（无需数据库服务器）
- **定时任务**: node-schedule
- **邮件发送**: nodemailer
- **AI 分析**: OpenCode 本地 / OpenRouter / Ollama
- **网页解析**: cheerio + axios

## 常见问题

**Q: 如何获取 QQ 邮箱授权码？**  
A: 登录 QQ 邮箱 → 设置 → 账户 → 开启 SMTP 服务 → 获取 16 位授权码（不是登录密码）

**Q: AI 分析失败怎么办？**  
A: 1) 检查 AI 服务是否配置正确；2) 点击"测试 AI 连通性"排查；3) 如使用 OpenRouter，确认 API Key 有效且有额度

**Q: 如何修改定时采集频率？**  
A: 在系统设置 → 采集间隔（分钟），默认 30 分钟

**Q: 如何修改邮件发送时间？**  
A: 在系统设置 → 定时发送时间，格式为逗号分隔的 24 小时制时间，如 `08:00,12:00,18:00`
